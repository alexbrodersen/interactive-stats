2 Programming

Much of what you do in this lab will require you to be able to program. It is okay if you don’t have any experience with this - most psychologists get their first experiences programming in graduate school. In our lab we primarily use four programming languages.

  1. Python - General Purpose Scripting Language
  2. R - Statistical Programming Language
  3. c++ - fast scientific computing
  4. SQL - Database programming language.
  5. Javascript - Used for Building Websites

The majority of this tutorial will be written in R. That is because R is a pretty user friendly programming language.

That means: - It’s difficult to break your whole computer because you messed something up while using R. - The syntax is mostly intuitive and easy to understand. - It’s pretty easy to learn another programming language once you have learned R.

Additionally, and we use it extensively for our data analysis in the lab (with very few exceptions).

Before you learn R, you must install it on your computer. For most people, this also means installing an integrated development environment called RStudio.

RStudio is an excellent choice for a text editor when learning R. It has many tools that make usering R easier.

2.1 Install R

2.1.1 Mac Users

  1. Open an internet browser and go to www.r-project.org.
  2. Click the “download R” link in the middle of the page under “Getting Started.”
  3. Select a CRAN location (a mirror site) and click the corresponding link.
  4. Click on the “Download R for (Mac) OS X” link at the top of the page.
  5. Click on the file containing the latest version of R under “Files.”
  6. Save the .pkg file, double-click it to open, and follow the installation instructions.

Now that R is installed, you need to download and install RStudio.

2.1.2 Windows Users

  1. Open an internet browser and go to www.r-project.org.
  2. Click the “download R” link in the middle of the page under “Getting Started.”
  3. Select a CRAN location (a mirror site) and click the corresponding link.
  4. Click on the “Download R for Windows” link at the top of the page.
  5. Click on the “install R for the first time” link at the top of the page.
  6. Click “Download R for Windows” and save the executable file somewhere on your computer. Run the .exe file and follow the installation instructions.

Now that R is installed, you need to download and install RStudio.

2.2 Install RStudio

2.2.1 Mac Users

  1. Go to www.rstudio.com and click on the “DOWNLOAD RSTUDIO DESKTOP” button towards the bottom of the page.
  2. Click on the version recommended for your system, or the latest Mac version, save the .dmg file on your computer, double-click it to open, and then drag and drop it to your applications folder.

2.2.2 Windows Users

  1. Go to www.rstudio.com and click on the “DOWNLOAD RSTUDIO DESKTOP” button towards the bottom of the page.
  2. Click on the version recommended for your system, or the latest Windows version, and save the executable file. Run the .exe file and follow the installation instructions.
